详细描述

常用 modbus 主机操作函数 

   API 形式如下：
   - agile_modbus_serialize_xxx    打包请求数据
   返回值:
       >0:请求数据长度
       其他:异常

   - agile_modbus_deserialize_xxx  解析响应数据
   返回值:
       >=0:对应功能码响应对象的长度(如 03 功能码，值代表寄存器个数)
       其他:异常 (-1：报文错误；其他：可根据 `-128 - $返回值` 得到异常码)
